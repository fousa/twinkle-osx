Twinkle

Make your Blink(1) change color on application change

Download this project as a .zip file Download this project as a tar.gz file

Blink(1)

I bought the Blink(1) device on a Kickstarter project. I hope to do more cool stuff with it!

App

Every time you change the focus to another application you Blink(1) device will change to the color you set in the preference panel. The app is OS X only.

Change the settings

You can start listening to an application by selecting the application in the list and selecting the Active checkbox. The color that is set will be the color of the Blink(1) device when you switch to the selected application.
